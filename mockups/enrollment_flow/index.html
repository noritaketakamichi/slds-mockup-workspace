Enrollment Flow Mockups

各オプションの画面遷移フローを選択してください。各フローは student-detail から始まり、割引設定画面で終わります。

Option A: マルチタグ割引（インライン）
商品ごとにタグベースの割引をインラインで設定するパターン。複数の割引タグを組み合わせた柔軟な割引構成が可能。
student-detail → application-detail → product-select → order-group-edit → product-add-change → discount-option-a
Option B: 割引ウォーターフォール
割引を段階的（ウォーターフォール形式）に適用するパターン。割引ステップの順序や条件を視覚的に確認・編集できる。
student-detail → application-detail → product-select → order-group-edit → product-add-change → discount-option-b
Option C: 自動検出 + 予測カード
適用可能な割引を自動検出し、予測結果をカード形式で表示するパターン。割引の比較・シミュレーションが容易。
student-detail → application-detail → product-select → order-group-edit → product-add-change → discount-option-c
