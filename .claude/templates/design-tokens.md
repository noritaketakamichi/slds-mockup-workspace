# SLDS デザイントークン リファレンス

## カラー

### ブランドカラー
- `--slds-g-color-brand-base-50`: プライマリブランドカラー (#0176d3)
- `--slds-g-color-brand-base-40`: ブランドダーク (#014486)
- `--slds-g-color-brand-base-60`: ブランドライト (#1b96ff)

### テキストカラー
- `--slds-g-color-neutral-base-10`: デフォルトテキスト (#181818)
- `--slds-g-color-neutral-base-50`: セカンダリテキスト (#706e6b)
- `--slds-g-color-neutral-base-70`: プレースホルダテキスト (#939393)

### 背景カラー
- `--slds-g-color-neutral-base-95`: ページ背景 (#f3f3f3)
- `--slds-g-color-neutral-base-100`: カード・モーダル背景 (#ffffff)

### ステータスカラー
- `--slds-g-color-success-base-50`: 成功 (#2e844a)
- `--slds-g-color-warning-base-50`: 警告 (#dd7a01)
- `--slds-g-color-error-base-50`: エラー (#ea001e)
- `--slds-g-color-info-base-50`: 情報 (#0176d3)

## スペーシング

- `slds-p-around_xx-small`: 0.25rem (4px)
- `slds-p-around_x-small`: 0.5rem (8px)
- `slds-p-around_small`: 0.75rem (12px)
- `slds-p-around_medium`: 1rem (16px)
- `slds-p-around_large`: 1.5rem (24px)
- `slds-p-around_x-large`: 2rem (32px)

## マージン

- `slds-m-top_small` / `slds-m-bottom_small`: 0.75rem
- `slds-m-top_medium` / `slds-m-bottom_medium`: 1rem
- `slds-m-top_large` / `slds-m-bottom_large`: 1.5rem
- `slds-m-left_x-small` / `slds-m-right_x-small`: 0.5rem

## フォントサイズ

- `slds-text-heading_large`: 1.5rem (24px)
- `slds-text-heading_medium`: 1.25rem (20px)
- `slds-text-heading_small`: 1rem (16px)
- `slds-text-body_regular`: 0.8125rem (13px)
- `slds-text-body_small`: 0.75rem (12px)

## グリッドシステム

```
slds-size_1-of-1    → 100%
slds-size_1-of-2    → 50%
slds-size_1-of-3    → 33.33%
slds-size_2-of-3    → 66.67%
slds-size_1-of-4    → 25%
slds-size_3-of-4    → 75%

slds-medium-size_*  → 768px以上で適用
slds-large-size_*   → 1024px以上で適用
```

## よく使うユーティリティクラス

- `slds-truncate`: テキストを1行で切り詰め
- `slds-text-align_center`: テキスト中央寄せ
- `slds-hide`: 要素を非表示
- `slds-show`: 要素を表示
- `slds-border_bottom`: 下線ボーダー
- `slds-box`: パディング付きボックス
